## コーパス

### 一般コーパス

- [DWDS Corpus](https://www.dwds.de/d/korpora) - 広範なテキストコーパスを持つドイツ語デジタル辞典。
- [DEREKO](https://www.ids-mannheim.de/digspra/kl/projekte/korpora/) - ドイツ語研究所によるドイツ語参照コーパス。
- [DeReKo-2019-I](https://www.ids-mannheim.de/digspra/kl/projekte/korpora/releases/) - ドイツ語参照コーパスのリリース版。
- [GerManC](https://www.llc.manchester.ac.uk/research/projects/germanc/) - 歴史的ドイツ語テキストのマンチェスターコーパス。
- [TIGER Corpus](https://www.ims.uni-stuttgart.de/forschung/ressourcen/korpora/tiger/) - 構文注釈付きドイツ語新聞テキストコーパス。

### 専門コーパス

- [TüBa-D/Z](https://uni-tuebingen.de/fakultaeten/philosophische-fakultaet/fachbereiche/neuphilologie/deutsches-seminar/abteilungen/computerlinguistik/ressourcen/corpora/tueba-dz/) - チュービンゲン書き言葉ドイツ語ツリーバンク。
- [C-WEP](https://www.cl.uzh.ch/de/texttechnologie/research/cwb/cwep.html) - 教養専門ドイツ語書き言葉コーパス。
- [SdeWaC](https://www.ims.uni-stuttgart.de/forschung/ressourcen/korpora/sdewac/) - .deドメインからのドイツ語ウェブコーパス。
- [DIRNDL Corpus](https://www.ims.uni-stuttgart.de/forschung/ressourcen/korpora/dirndl/) - 言語学のためのラジオニュース談話情報データベース。

### 歴史的コーパス

- [DTA](https://www.deutschestextarchiv.de/) - ドイツ語テキストアーカイブ。
- [GerManC-GS](https://www.llc.manchester.ac.uk/research/projects/germanc/) - 初期近代ドイツ語のジャンルバランスコーパス。
- [Referenzkorpus Altdeutsch](https://www.deutschdiachrondigital.de/) - 古高ドイツ語と古サクソン語の参照コーパス。
- [Referenzkorpus Mittelhochdeutsch](https://www.linguistics.ruhr-uni-bochum.de/rem/) - 中高ドイツ語参照コーパス。

## ツール

### コーパス分析

- [CQPweb](https://cwb.sourceforge.io/cqpweb.php) - ウェブベースのコーパス分析ツール。
- [SketchEngine](https://www.sketchengine.eu/) - コーパス分析と用語抽出ツール。
- [AntConc](https://www.laurenceanthony.net/software/antconc/) - コーパス言語学のためのフリーウェアコンコーダンスプログラム。
- [NoSketchEngine](https://nlp.fi.muni.cz/trac/noske) - オープンソースのコーパス検索システム。

### 言語学習

- [German-English Dictionary App](https://german-english-dictionary.devtheworld.jp/) - 科学的に効率的な現代的学習のため。
- [Linguee](https://www.linguee.de/) - コーパス例文付き辞書・翻訳検索エンジン。
- [PONS](https://de.pons.com/) - 例文付きオンライン辞書。

### テキスト処理

- [spaCy German Models](https://spacy.io/models/de) - 産業用途の自然言語処理モデル。
- [SoMaJo](https://github.com/tsproisl/SoMaJo) - ドイツ語テキストのためのトークナイザーと文分割器。
- [Germalemma](https://github.com/WZBSocialScienceCenter/germalemma) - ドイツ語テキストの見出し語化。
- [German NLP](https://github.com/adbar/German-NLP) - ドイツ語のためのオープンアクセス/オープンソースNLPリソースの厳選リスト。

## 学術テキスト

- [OPUS](http://opus.nlpl.eu/) - 学術テキストを含むオープン並列コーパス。
- [Project Gutenberg (German)](https://www.gutenberg.org/browse/languages/de) - ドイツ語の無料電子書籍。
- [Freiburg Corpus](https://www.freiburg-corpus.de/) - 話し言葉ドイツ語の学術コーパス。
- [ZAS Database of Clause-Embedding Predicates](https://www.zas.gwz-berlin.de/en/research/databases.html) - ドイツ語述語のための言語学データベース。
